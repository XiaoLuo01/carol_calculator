# carol_calculator
用 Electron 写的一个简单的桌面计算器应用 demo

## 拉取项目
```
git clone https://github.com/XiaoLuo01/carol_calculator.git
```

## 安装依赖
```
npm install
```

## 启动项目
```
npm start
```

## 打包成可执行文件
```js
// window 系统
npm run packageWin

// mac 系统
npm run packageMac
```

## 打包成安装包
```js
// window 系统
npm run buildwin

// mac 系统
npm run buildmac
```
